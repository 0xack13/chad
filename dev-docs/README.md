## Building from source
The following are required to build the Chad interpreter from source

* gcc
* rustc
* cargo
* make
* nasm


1. Clone the repository

`git clone https://github.com/0xhiro/chad`

`cd chad`

2. Compile

`make all`

3. Install

`make install`

4. Run a script

`chad run hello-world.chad`
